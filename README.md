# **Human Resource Management**

> HR management console application

## Environmental requirements

✅ Any IDE or editor that can write C#

✅ .NET SDK 8.0

✅ Windows 10 or later

## Getting Started

_First, Go to project folder to start_

```bash
cd human-resource-management
```

_Run project_

```bash
dotnet run
```
